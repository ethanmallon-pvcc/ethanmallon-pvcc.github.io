Chapter 4 Terms

Term Definition
Network Classifications
LAN Local Area Network, typically conists of a collection of computers in a single building or complex
MAN Metropolitan Area Network, and intermediate sized network that typically covers a local community
WAN Wide Area Network, a network linking devices over great distances such as other cities and countries
Network Models
Bus Network A network in which all machines are connected to a common communication line, itself called a "bus"
Star Network A network in which one machine serves a a focal point for others to connect to
Protocols
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detection, a protocol dictating each machine on the bus receives a message, then when transmitting wait for the bus to be silent. If another begins to transmit, both stop for a random period of time before reattempting
CSMA/CA Wifi Carrier Sense, Multiple Access with Collision Avoidance, like CSMA/CD except works to prevent transmissions collisions from occuring rather than just detecting them
Combining Networks
Repeater A device that passes signals back and forth between buses
Bridge Similar to a repeater, but filters the messages it tranfers to determine if the recipient bus possesses the addressed device
Switch A bridge that has mutliple connections, bridging more than 2 buses
Router Special purpose computers used for forwarding messages
Client/Server Model
P2P: Peer-to-Peer A model involving processes that provide and receive services from each other
Distributed System Systems that consit of software that execute as processes on different computers
Cloud Computing A style of distributed systems where pools of shared computers on a network can be allocared for client use as needed
Internet Addressing
Tier 1 Highest tier Internet Service Providers consisting of high-speed, high-capacity, and international WANs
Tier 2 Mid-tier ISPs that are more regional and less potent
Tier 3 Lowest tier "access" ISPs, essentially an independent internet operated by a single authority supplying Internet access to individual homes and businesses
IP Address
ICANN Internet Corporation for Assigned Names and Numbers, a nonprofit established to coordinate the Internet's operation
DNS Domain Name System, an Internet-wide directory system based off of numerous "name" servors
TLD: Top-Level Domain Refers to the common suffixes found at the end of URLs, such as .edu, .com, etc.
TLDs Top Level Domains, the suffixes found at the end of a website's URL that indicates what type of website it is.
Internet Protocols
FTP File Transfer Protocol, an application for accessing another computer from a significant distance
HTTP Hyper Text Transfer Protocol, a protocol uses to transfer Hypertext, or linked documents
SMTP Simple Mail Transfer Protocol, a protocol uses for email transfer over a network
World Wide Web
URL Uniform Resource Locater, contains the information needed by a browser to contact the proper server and request the desired document.
Client-Side Activities As the name suggests, activities performed exclusively by the client
Server-Side Activities Activities performed by the server itself
Internet Four-Layer Software
Application Layer Where messages originate
Transport Layer Used by the application layer to send and receive messages by "packaging" and numbering them for later reconstructions
Network Layer Takes the message packages from the transport layer, treating them as individual and unrelated messages until the reach the transport layer of their destination
Link Layer Responsible for package transfer and dealing with communication details particular to the computers individual network
Internet is:
Packet-Switching Network A network in which data is sent by breaking the data into blocks of bits called packets which contain the transmitted data and it's control information
Fault Tolerant Programs designed to continue operation despite error, exception, or invalid input,rather than simply crashing
Redundant Having more than one of something in order the increase reliablility in the event that if one things fails another will succeed
Cybersecurity
Malware Malicious software often used in cyberattacks
Virus Software that infects computers by inserting itself into programs that are already in the machine
Worm An autonomous program that transfers through a network, takes residency in a computer and forwards copies of itself to other computers
Trojan Horse A program that enters the computer system disquised as a desired program, and require the victim to purposefully import it
Spyware A form of malware dedicated to secretly collecting various imformation regarding its host computer, such as its activities and residence
Phishing As opposed to spyware, it collects information by outright asking for it by providing a bait to trick the victim into giving information
DDoS Attack Distributed Denial of Service, a method of attack which uses enormous amounts of messages to overwhelm the target server, specifically DDoS refers to when this is distributed across many hosts
Protection
Firewall A prevention technique that essentially functions as a "bouncer", only allowing properly identified message through and/or blocking certain message IDs
Spam Filter Firewalls designed to block unwanted emails in particular. Do to the blurry line between desired emails and spam these filters are rather sophisticated
Proxy Server A software unit that acts as an intermediary between client and server with the goal of blocking adverse server actions from the client
Antivirus Software Software used to detect and remove viruses and infections
Cryptography The study of sending and receiving messages securely despite present adversaries
Public Key Encryption A commonly used encryption method due to the enhanced security caused by its use
