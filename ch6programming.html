Chapter 6 Terms

Term Definition
Machine Language An encoding system/program using intruction sequences encoded as numeric digits used in computers.
Debugging The location and correction of program errors.
Assembler Programs developed to convert mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system used to represent programs.
Translator A program designed to convert high-level primitives into machine language programs
Compiler A form of translator that in addition to translating language, compiled instructions into short sequences in order to simulate the activity requested by a high-level primitive.
Interpreter Similar to translators, except they execute instructions as they are translated rahter than recording them for future use.
Cross-Platform Software Software independent of an operating system's design or machine's hardware in order to be executable through a network.
Natural Language Spoken language (English, German, etc.) that developed without formal gramatical analysis.
Formal language Written language (programming languages) precisely grammatically defined.
Machine Independence Programming languages that do not rely on the characteristics of a specific machine.
High-Level Programming Language Provide more abstractions for the programmer and make reading and writing programmings simpler.
Object Oriented Programming (OOP) Programming where software is viewed asa collection of units (objects), with each capable of performing immediately related actions for itself and at the request of other objects.
Scripting language Languages used to perform admin tasks rath than develop complex programs.
Data Types: Integer Refers to numeric data consisting of whole numbers.
Data Types: Float Numeric data that contains values other than whole numbers.
Data Types: String A sequence of data in character form.
Primitive Data Type A set of basic data types used to construct other data types.
Data Structure The conceptual shape or arrangement of data.
Python Structure: List Used to store multiple items in a single variable. Uses square brackets []
Python Structure: Tuple Like a list, but ordered, unchangeable, and written with round brackets ()
Python Structure: Set Same principles as a tuple, in addition to being unindexed. Written using culy brackets {}
Python Structure: Dictionary Store data values in value pairs. The collection is ordered, changeable, and does not allow duplicates. Uses curly brackets with keys and values.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function and can only be referenced to within that function.
Global Variable Variables not restricted to a particular part of a program.
Source Program The program in its original form.
Object Program The translated version of the source program.
